# LÚMINA - Tienda Online de Joyería

Monorepo para una tienda premium de joyería artesanal de lujo.

## 📦 Estructura del Monorepo

```
joyas-web/
├── apps/
│   └── web/        # Frontend Next.js
├── packages/       # Librerías compartidas (futuro)
└── ...
```

## 🚀 Inicio Rápido

### Instalar dependencias del monorepo

```bash
npm install
```

### Desarrollar

```bash
npm run dev
```

Esto iniciará automáticamente la aplicación web en `http://localhost:3000`

### Compilar para producción

```bash
npm run build
```

## 🎨 Diseño

- **Estilo**: Minimalista y elegante
- **Paleta**: Blanco, crema, dorado y gris oscuro
- **Tipografía**: Playfair Display (headings) y Montserrat (body)
- **Filosofía**: Lujo accesible, artesanía premium

## 🛠️ Tech Stack

### Frontend

- Next.js 14
- React 18
- TailwindCSS
- TypeScript

### Próximamente Backend

- Node.js / Express
- Prisma ORM
- PostgreSQL
- JWT Authentication
